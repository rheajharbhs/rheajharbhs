HTML TEXT INPUT Tag

The <input> tag specifies an input field where the user can enter data. The <input> element is the most important form element.
Example:
Student Name:
Code snippet
<label>Student Name:</label> <input type="text">

HTML RADIO Button

The <input> tag specifies an input field where the user can enter data. The <input> element is the most important form element.
Example:
WEIGHTED: YESNO
Code snippet
<label>WEIGHTED:</label> <input type="radio" name="radio1">YES</input> <input type="radio" name="radio1">NO</input>

HTML SELECT Tag

The <select> tag specifies an input field from where the user can select a value from a list of options. The <select> element helps in restricted input and avoids user input errors.
Example:
STATE:
Code snippet
<label>STATE:</label> <select> <option value="MO">Missouri</option> <option value="IL">Illinois</option> </select>

HTML BUTTON INPUT Tag

The <input> tag specifies an input field where the user can enter data. The <input> element can be defined as a button and capture different action user wants to perform.
Example:
Code snippet
<input type="button" value="Add to Cart"/> <input type="button" value="Go to Cart"/> <input type="button" value="Place Your Order"/>
